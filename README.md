# WoW AI Class Helper

An AI-powered web application to help World of Warcraft players master their class. Get insights on rotations, talent builds, addons, and dungeon strategies generated by Gemini.

## Quick Start

**Prerequisites**: Node.js 18+

```bash
# 1. Install dependencies
npm install

# 2. Set up environment
echo "GEMINI_API_KEY=your_key_here" > .env.local

# 3. Run development server
npm run dev
```
